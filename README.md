## 開発環境

- 言語 : Ruby2.5.7
- フレームワーク : Rails5.2.4
- バージョン管理 : Github
- DB : mysql2、sqlite3

## 実装した機能

### ユーザー機能

- ユーザー登録機能(devise)
- ログイン機能(devise)
- 投稿機能(modal)
- 投稿一覧機能
- コメント機能(非同期:jquery)
- 検索機能
- カテゴリー機能
